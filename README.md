# 🛍️ Black Friday Sales Analysis Dashboard

## 📌 Project Overview
This project presents an **interactive Black Friday Sales Analysis Dashboard** built using retail transaction data. The dashboard uncovers customer purchasing behavior, product trends, city-wise sales performance, occupation-based spending, and demographic insights.  

It helps businesses understand consumer patterns during Black Friday shopping events and supports **data-driven decision-making** for marketing and inventory planning.

---

## 🎯 Objectives
- Analyze customer buying behavior  
- Identify top-performing product categories  
- Understand demographic spending patterns  
- Discover city-wise sales trends  
- Improve business decision-making using data visualization  

---

## ✨ Features
- Gender-wise purchase analysis  
- Age group purchase distribution  
- Occupation-based sales analysis  
- City category comparison  
- Product category performance  
- Top-selling products analysis  
- Marital status vs purchase analysis  
- Customer distribution analysis  
- Purchase trend visualizations  
- Interactive business intelligence dashboard  

---

## 📊 Dashboard Insights
- Male customers contributed higher overall purchases compared to female customers.  
- Customers aged **26–35** generated the highest sales.  
- **City Category B** recorded the maximum purchases.  
- Certain occupations showed significantly higher spending patterns.  
- **Product Category 1** contained the top-selling products.  
- Average purchase value varied across gender and age groups.
